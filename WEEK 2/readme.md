This directory contains the solution files for the Week 2 exercises of the Cognizant Deepskilling program, focusing on Spring Core and Maven project management.

Overview :- 
These exercises help you understand:
Dependency management using Maven.
Configuring Spring Core beans.
Inversion of Control (IoC) and Dependency Injection (DI) principles.

Contents:-
SpringCoreMaven_Solutions.zip: A compressed archive containing the complete source code and configuration files for the Week 2 tasks.
