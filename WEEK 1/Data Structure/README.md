# Week 1: Data Structures and Design Patterns
This directory contains the solutions and exercises completed during the first week of the Cognizant Digital Nurture 5.0 program.
## Overview
The focus of this week was on mastering fundamental Data Structures and implementing robust Design Patterns to solve real-world software problems.
## Exercises
### Data Structures
* **Inventory Management System:** Core logic for managing stock and assets using a `HashMap` for \(O(1)\) average time complexity for insertion, update, and deletion operations.
* **E-commerce Platform Search:** Implementation of linear search \(O(n)\) and binary search \(O(\log n)\) to search products, demonstrating search performance differences.
* **Sorting Customer Orders:** Application of sorting algorithms (Bubble Sort \(O(n^2)\) and Quick Sort \(O(n \log n)\)) to organize transaction data by price.
* **Employee Management System:** CRUD operations using a custom array-based structure with memory shift operations.
* **Task Management System:** Efficient task handling and tracking using a custom singly linked list implementation.
* **Library Management System:** Book tracking and inventory search using binary search on sorted book titles.
* **Financial Forecasting:** Implementation of recursive and iterative growth calculations for predicting future values.
